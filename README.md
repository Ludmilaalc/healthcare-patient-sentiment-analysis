Patient Sentiment Analysis in Healthcare

📌 Context

This project analyzes patient perception based on public Google Reviews data. The goal is to understand satisfaction patterns, identify critical issues in the patient journey, and support data-driven decision-making.

🎯 Objective
	•	Analyze sentiment (Positive, Neutral, Negative)
	•	Evaluate trends over time
	•	Identify operational and reputational risks
	•	Generate insights to improve patient experience

🧠 Methodology

The analysis was structured in the following steps:
	1.	Data collection from Google Reviews
	2.	Data cleaning and preprocessing
	3.	Sentiment classification using NLP
	4.	Sentiment distribution analysis (%)
	5.	Time series analysis
	6.	Text analysis (Top 10 positive and negative words)
	7.	Insight generation and interpretation

📊 Key Insights
	•	High neutral sentiment suggests a functional but not memorable experience
	•	Negative reviews indicate operational gaps in service delivery
	•	Variation across units suggests inconsistency in service quality
	•	Strong opportunity to improve patient experience and satisfaction

🚀 Business Impact

This analysis enables:
	•	Identification of operational bottlenecks
	•	Improvement of patient experience
	•	Data-driven decision making
	•	Reputation risk mitigation

📁 Project Structure
	•	notebooks/ → Individual analysis per unit (files named in Portuguese)
	•	images/ → Dashboards and visual insights
	•	reports/ → PDF summaries of analysis

🌍 Language Note

The analysis notebooks are written in Portuguese, while the project documentation is provided in English.

🔒 Disclaimer

All data has been anonymized. Units are represented as Unit A, B, C, D, and E.
