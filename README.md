# 📊 Patient Sentiment Analysis in Healthcare

## 📌 Context
This project analyzes patient perception based on public Google Reviews data. The goal is to understand satisfaction patterns, identify critical issues in the patient journey, and support data-driven decisions.

## 🎯 Objective
- Analyze sentiment (Positive, Neutral, Negative)
- Evaluate trends over time
- Identify operational and reputational risks
- Generate insights for improving patient experience

## 🧠 Methodology
The analysis was structured in the following steps:

1. Data collection from Google Reviews
2. Data cleaning and preprocessing
3. Sentiment classification using NLP
4. Percentage calculation by sentiment
5. Time series analysis
6. Text analysis (Top 10 positive and negative words)
7. Interpretation of results

## 📊 Key Insights
- High neutral sentiment suggests functional but not memorable experience
- Negative reviews indicate operational gaps
- Variation between units suggests inconsistent service quality
- Opportunity to improve patient experience and satisfaction

## 📁 Project Structure
- notebooks/ → analysis per unit
- images/ → dashboards and visualizations

## 🔒 Disclaimer
All data has been anonymized. Units are represented as Unit A, B, C, D, and E.

## 🛠️ Tools
- Python
- Pandas
- NLP
- Power BI
